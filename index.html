JavaScript와 HTML의 input

HTML에서는 "input 태그를 제공합니다."

이 input 태그는 아래와 같이 작성됩니다.

<input type="input 의 타입" value="초기저장값" />

type 속성에는 input의 종류를 정해진 키워드에 따라 입력합니다.

input의 타입

Input 타입 설명 예시
text 일반 텍스트 입력란
password 비밀번호 입력란 (텍스트가 마스킹됨)
email 이메일 입력란 (이메일 형식 검사)
number 숫자 입력란 (숫자만 입력 가능)
date 날짜 입력란
checkbox 체크 박스
radio 라디오 버튼 (그룹에서 하나 선택) 남성 여성
file 파일 업로드 버튼
submit 전송 버튼
reset 초기화 버튼
button 일반 버튼 (onclick="" 의 "" 사이에 버튼 클릭시 실행되는 js 코드 입력)
color 색상을 입력 (value 에는 hex(#000000) 이 저장됨

이렇게 사용자에게 정보를 입력받으면, input 요소는 이를 "value" 에 저장하며, js 에서 아래와 같이 접근할 수 있습니다.

<input type="text" id="a" value="고라니">
  <script>
    let user_input = document.querySelector("#a").value;
    // user_input 변수는 "a" ID를 가진 input 요소의 value 값인 "고라니"를 할당받습니다.
    console.log(user_input); // 콘솔에 출력하여 확인할 수 있습니다.
  </script>

    function changeColor() {
      const text = document.querySelector("#text");
      const color = document.querySelector("#color").value;
      text.style.color = color;
    }
  </script>
